Data Schema

All cities included in the California Crime Dashboard are standardized into a common schema. Each row represents one reported crime incident, with consistent columns across:

  • Los Angeles
  • San Diego
  • San Francisco

This ensures analytical consistency for trends, forecasting, and geographic comparison.

üîç Notes on Data Cleaning

1. Geographic Filters

Rows are removed if coordinates are:

  • Missing
  • Invalid (0,0)
  • Outside realistic California bounds (lat 32‚Äì38, lon ‚Äì125 to ‚Äì114)

2. Time Features

Generated from timestamps:

  • year, month, hour
  • time_bin
  • day_of_week
  • season

3. Crime Category Standardization

Crime codes from each city are mapped into unified categories such as:

  • Property Crime
  • Violent Crime
  • Vehicle Crime
  • Theft
  • Vandalism
  • Fraud
  • Other Offenses

4. Victim Information

When available:

  • Victim age
  • Victim sex
  • Victim descent
